# pony的个人博客

#### 介绍
这个博客是我下个阶段要经常维护的一个项目，有什么技术上的新特性会和大家即使分享出来的，这个也是我对vue2.0一次个人项目，vue技术栈的大集合。

#### 软件架构
使用的技术有vue2.0,vuex状态管理器，路由管理系统vueRouter,数据请求使用了axios并且做出了封装。


#### 安装教程

1.  yarn install
2.  yarn serve
